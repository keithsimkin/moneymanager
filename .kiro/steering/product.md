# Product Overview

cashflow.pilot is a personal finance management application that helps users track their financial activities. The app provides comprehensive tools for managing accounts, transactions, budgets, and financial goals.

## Core Features

- **Account Management**: Support for checking, savings, credit, and investment accounts
- **Transaction Tracking**: Record income and expenses with categorization
- **Budget Planning**: Set and monitor spending limits by category (weekly/monthly/yearly)
- **Goal Setting**: Track progress toward financial goals with deadlines
- **Recurring Transactions**: Automate regular income and expenses
- **Analytics**: Visualize spending patterns, trends, and budget progress
- **Data Portability**: Export and import financial data (JSON format)

## Data Storage

All data is stored locally in the browser using localStorage. No backend server or external APIs are used.
